## Custom code boilerplate

Install the dependencies

```bash
npm install
```

#### Running it locally

Regular layout
```
npm start
```

Longform standard
```
npm run start-ls
```

Longform Visual
```
npm run start-lv
```

Navigate to [http://0.0.0.0:5555](http://0.0.0.0:5555). You should see your app running. Edit a component file in `src`, save it and see the changes in the browser.
